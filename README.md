## 💻 Getting started

### Requirements

- [Node.js](https://nodejs.org/en/)

**Clone the project and access the folder**

```bash
git clone https://github.com/luancardosoti/robots-battle.git && cd robots-battle
```

**Follow the steps below**

```bash
# Install the dependencies
$ npm i

# Run json-server
$ npm run server

# Run project
$ npm run dev
```
